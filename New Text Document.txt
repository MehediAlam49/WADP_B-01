4th step> activate (for activate env)

5th step> pip install Django (for install Django)
6th step> back to root folder> cd ../.. (for back root folder)
7th step> django-admin startproject myProject (create project)
8th step> cd myProject (for inside project)
9th step>python manage.py runserver (run server for output)
